# AGENTS.md — @radiance/gl

`@radiance/gl` is a WebGL2 library structured as a pnpm monorepo with the following packages:

- `packages/gl` — the main library package → see [packages/gl/AGENTS.md](packages/gl/AGENTS.md)
- `packages/helpers` — a utility package for shared helpers that don't depend on WebGL
- `docs/` — the documentation website → see [docs/AGENTS.md](docs/AGENTS.md)
