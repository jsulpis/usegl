## Usage

<Code lang="js" code={`
import { useWebGLCanvas } from "usegl";

useWebGLCanvas({
		canvas: document.querySelector("canvas"),
		fragment: /* glsl */ \`
			varying vec2 vUv;
			uniform float uTime;

			void main() {
				gl_FragColor = vec4(vUv, sin(uTime) / 2. + .5, 1.);
			}
		\`
});
`}  />
